# 🎯 Number Guessing Game

A responsive web-based Number Guessing Game built using **Flask**, **Python**, **HTML**, **CSS**, and **Bootstrap**.

---

## 🚀 Features

- Three difficulty levels
- Session-based gameplay
- Score tracking
- Attempts counter
- Too High / Too Low hints
- Play Again option
- Change Difficulty
- Responsive Bootstrap UI

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5

## ▶️ Installation

Clone the repository

```bash
git clone <repository-link>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
